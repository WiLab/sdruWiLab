setupsdru

Set up MATLAB® to work with Support Package for USRP® Radio

Syntax

setupsdru

Description

setupsdru modifies the MATLAB path and sets up architecture-specific environment variables. This must be done in every MATLAB session. You can automate this step in future sessions by adding setupsdru to your startup.m file. If you prefer to do this setup manually in each session, you can use the "Add SDRu" shortcut that has been provided on the MATLAB shortcut bar.

Examples

Set up environment for Support Package for USRP® Radio

>>setupsdru('C:\MATLAB\Targets\R2012b\commusrp')
