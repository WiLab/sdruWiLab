New features and changes introduced in this version are:

SDRu System Objects

This software includes UHDTM-based SDRu Transmitter and Receiver System objects for use with MATLAB. These System objects enable communication with USRP® hardware, allowing for development work in software-defined radio applications. For more information, see the comm.SDRuTransmitter and comm.SDRuReceiver help pages.

SDRu Blocks

This software includes UHDTM-base SDRu Transmitter and Receiver blocks for use with Simulink. These blocks enable communication with USRP® hardware, allowing for development work in software-defined radio applications. For more information, see the SDRu Transmitter and SDRu Receiver block help pages.

New Examples

This software includes MATLAB examples which use USRP® hardware with the SDRu Transmitter and SDRu Receiver System objects in various applications.

  • FM Monophonic Receiver with USRP® Hardware: To open this example file, enter edit sdruFMMonoReceiver at the MATLAB command line.

  • FM Stereo Receiver with USRP® Hardware: To open this example file, enter edit sdruFMStereoReceiver at the MATLAB command line.

  • FRS/GMRS Walkie-Talkie Transmitter with USRP® Hardware: To open this example file, enter edit sdruFRSGMRSTransmitter at the MATLAB command line.

  • FRS/GMRS Walkie-Talkie Receiver with USRP® Hardware: To open this example file, enter edit sdruFRSGMRSReceiver at the MATLAB command line.

  • Getting Started with SDRu: To open this example file, enter edit sdrugettingstarted at the MATLAB command line.

This software also includes Simulink examples which use the SDRu Transmitter and SDRu Receiver blocks in various applications.

  • FM Monophonic Receiver with USRP® Hardware: To open this example, enter sdruFMMono at the MATLAB command line.

  • FM Stereo Receiver with USRP® Hardware: To open this example, enter sdruFMStereo at the MATLAB command line.

  • FRS/GMRS Walkie-Talkie Transmitter Using USRP® Hardware: To open this example, enter sdruFRSGMRSTx at the MATLAB command line.

  • FRS/GMRS Walkie-Talkie Receiver Using USRP® Hardware: To open this example, enter sdruFRSGMRSRx at the MATLAB command line.

  • Getting Started with SDRu: To open this example file, enter sdrugettingstarted at the MATLAB command line.

Known Issues

Limitation When Using XCVR2450 Daughterboard

MATLAB will abort if you disconnect a USRP® device with an XCVR2450 daughterboard under one of the following conditions:

  • You are simulating a model that contains an SDRu Simulink block and the block dialog is open, or

  • You are using an SDRu System objectTM that is communicating with the USRP® hardware.

Full-Duplex Support

Communications with USRP® only supports half-duplex applications. MATLAB and Simulink applications must be receive-only or transmit-only. Full-duplex is not supported.

Multiple-Input Multiple-Output (MIMO) Support

Use Communications with USRP® in single-input, single output (SISO) applications. This software does not support the transmission of multiple, synchronized data streams to and from multiple USRP® transmitters and receivers.

Antenna Configuration

Antenna configurations for daughterboards remain in their default settings and cannot be changed. For more information, see Selecting an RF Daughterboard on the Ettus ResearchTM web site for details.
