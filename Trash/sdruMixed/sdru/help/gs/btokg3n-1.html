UHDTM driver cannot set thread priorities

Problem

You may see a warning stating that the UHDTM driver was not able to set the thread priority.

Back to Top

Possible Solution

This is a harmless error. You can get more information on this subject at http://files.ettus.com/uhd_docs/manual/html/general.html#thread-priority-scheduling.
