info

Class: comm.SDRuTransmitter
Package: comm

Display information about SDRu Transmitter System objectTM

Syntax

S = info(OBJ)

Description

S = info(OBJ) returns a structure, S, containing characteristic information for the System objectTM, OBJ. If OBJ has no characteristic information, S is empty. If OBJ has characteristic information, the fields of S vary depending on OBJ. For object specific details, refer to the help on the infoImpl method of that object.
