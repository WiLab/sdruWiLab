Firmware for USRP® radio is incompatible with host build

Problem

The function findsdru gives the following warning message:

Warning: The firmware build for the USRP radio at '192.168.10.2' is not compatible with the host code build. Update the USRP radio firmware to version '003.002.003-vendor'. Refer to Troubleshooting section of Getting Started document for instructions."

The firmware installed on the USRP® radio is incompatible with the software version of the support package.

Back to Top

Possible Solution

Update the firmware for your USRP® radio. See Update USRP® Radio Support Package Firmware.
