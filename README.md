sdruWiLab
========

Wireless communication system based on an OFDM implemented in MATLAB for USRP radios

__Complete:__
Modules
 - CRC
 - Custom messages
 - Class based control of tx and rx
 - Energy Detection
 - BCH Codes
 - OFDMA Example
