probesdru

Provides detailed USRP® radio information

Syntax

I=probesdru
I=probesdru(IPADDRESS)
[I,S]=probesdru

Description

I=probesdru returns detailed information about the USRP® radio connected to the host computer. The information is returned in argument I as a string. If there is more than one USRP® radio connected to the host computer, function probesdru returns radio information for the first discovered radio. Function probesdru works by calling theUHDTM application uhd_usrp_probe, provided by Ettus ResearchTM, as a system command and returns the command output.

I=probesdru(IPADDRESS) returns detailed information on the USRP® radio at IP address IPADDRESS. IPADDRESS is a dotted quad string, for example, "192.168.10.2".

[I,S]=probesdru returns the status of the system command execution in S. A non-zero S value indicates an error.

This function will return radio information for the first radio it finds. If you have more than one radio, you might want to use probesdru(IPADDRESS) instead.

 

Get detailed information on the USRP® radio located at IP address 192.168.10.2.

>>probesdru('192.168.10.2')
 

Get a list of all USRP® radios connected to the host computer and retrieve radio information for each one, using functions findsdru and probesdru.

rawDeviceList = findsdru();

for p=1:length(rawDeviceList)
    Info{p} = probesdru(rawDeviceList(p).IPAddress);
    disp(['Info for device ' num2str(p) ':']);
    disp(Info{p});
end
