getSDRuDriverVersion

Report UHDTM version number for the host

Syntax

A = getSDRuDriverVersion()

Description

A = getSDRuDriverVersion() returns the version number of the UHDTM software installed on the host computer.

Examples

Get the version number of the UHDTM software installed on the host computer.

>> getSDRuDriverVersion()    
ans =

003.004.002-vendor
