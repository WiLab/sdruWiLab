Product Overview

With Communications System Toolbox™ Support Package for USRP® Radio, MATLAB® and Simulink® connect to USRP® software-defined radios (SDR) from Ettus ResearchTM to provide a radio-in-the-loop design and modeling environment. With this support package, Communications System Toolbox, and a USRP® radio, you can design and verify practical SDR systems starting with Release 2011a of MATLAB and Simulink. This support package does not require GNU Radio open-source software.

Communications System Toolbox Support Package for USRP® Radio includes:

  • Use of USRP® as a standalone peripheral for live RF data I/O, including:

    • Functions and System objects for connecting MATLAB to UHDTM-based USRP2® radios

    • Blocks for connecting Simulink to UHDTM-based USRP® radios

  • Support for optional HDL code generation to customize USRP® N210 hardware, which requires HDL Coder™.

  • Examples to assist you:

    • QPSK transmitter and receiver

    • FM mono and stereo receivers

    • FRS/GMRS transmitter and receiver

    • And more

Communications System Toolbox Support Package for USRP® Radio has been tested on N210 and USRP2® devices using supported daughterboards (see the product Release Notes for a full list). Support may also extend to other UHDTM-based radios and daughterboards from Ettus ResearchTM.

About Documentation for Communications System Toolbox Support Package for USRP® Radio

  • From within MATLAB, enter the following command:

    >help sdru

    This command provides access to HTML pages for this Getting Started Guide, User Guide, examples, block and System objectTM help, and more.

  • In the Getting Started section, you can find more information for the following topics:

    • How to configure your host computer to communicate with the USRP® radio

    • How to discover connected radios and troubleshoot connectivity problems

    • How to update firmware

  • After you have finished your host setup and established connectivity between the host and the radio hardware, take a look at one of the examples for the Simulink block or a System objectTM:

    • FM Monophonic Receiver with USRP® Hardware - Simulink

    • FM Monophonic Receiver with USRP® Hardware - MATLAB

    These examples demonstrate the receiver functionality of Communications System Toolbox Support Package for USRP® Radio. When you are ready to begin, select a workflow and begin with the Determining Host-Hardware Connection Performance tutorial.
