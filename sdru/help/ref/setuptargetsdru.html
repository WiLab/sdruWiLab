setuptargetsdru

Downloads the FPGA interface code for USRP® hardware

Syntax

setuptargetsdru(DESTFOLDER)
SUCCESS = setuptargetsdru(DESTFOLDER)

Description

setuptargetsdru(DESTFOLDER) downloads the FPGA interface code from the UHDTM mirror site to the destination folder specified by DESTFOLDER.

SUCCESS = setuptargetsdru(DESTFOLDER) returns the status of the download in SUCCESS. If successful, returns the success status 1. In case of failure, SUCCESS contains success status 0.
